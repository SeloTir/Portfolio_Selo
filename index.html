Clash of Clan Tower Defense

Année : 2023

Contexte

Ce projet avait pour objectif de réaliser un tower defense en 2D en trio sur une durée de 6 semaines. Le but du jeu est de repousser des ennemis qui viennent dans différente vague à l'aide de nos défenses qu’il faudra placer stratégiquement.

Pour la gestion de notre projet, on nous a imposé l'utilisation de trello et découpé notre code en plusieurs itérations que l’on rendait tous les 10 jours.

Lien du projet : Clash of Clan Tower Defense

Description du jeu

Vous êtes le chef d'un puissant village et vous devez le défendre contre les attaques incessantes des hordes d'ennemis.

Votre objectif est de construire et de positionner stratégiquement des tours de défense pour repousser les assauts des ennemis qui deviennent de plus en plus fort après chaque vague.

Attention néanmoins à dépenser votre argent méticuleusement lors de la pose de défenses. Bien que vous pouvez les revendre, elles ne vous rapporteront pas autant que le prix a lesquelles vous les avez achetées !

Outils utilisé
  • Java
  • SceneBuilder
  • Github
  • Trello

Z-Hellda

Année : 2024

Contexte

Projet réalisé en trio en utilisant un peu de méthode agile sur une période de 7 semaines. Avec l'architecture MVC (modèle vue contrôleur) nous devions réaliser un petit zelda en 2D avec un personnage qui peut interagir, attaquer, se déplacer ou parler.

Pour la gestion de notre projet, nous avons utilisé github et trello, puis notre projet a été répartie en plusieurs sprints réalisés à peu près tous les 10 jours.

Lien du projet : Z-Hellda

Description du jeu

Entrez dans un monde qui mélange univers post-apocalyptique, enfer et Zelda. Votre but est de survivre dans ce monde chaotique et de vaincre le boss final qui arrivera après les vagues d’ennemis.

Vous disposerez pour cela de plusieurs types d’armes de courtes ou longues distances ainsi que des items consommables disponible dans les coffres ou sur le corps des ennemis qui influeront sur l’état du héros ou de ces armes.

Attention néanmoins à bien gérer votre inventaire et à utiliser au bon moment vos items consommables afin de ne pas vous retrouvez dans une mauvaise passe !

Outils utilisé
  • Java
  • CSS
  • SceneBuilder
  • Github
  • Trello

Bataille navale

Année : 2022

Contexte

Projet réalisé en duo sur une durée de 5 semaines. Réalisation d’un jeu de plateau sans le concept de programmation objet et avec comme seul interface le terminal. Utilisation de Github.

A la fin de notre projet, nous devions rendre des tests JUnit ainsi qu’une petite documentation.

Lien du projet : Bataille navale

Description du jeu

Deux joueurs s'affrontent l’un contre l’autre. Chaque joueur possède 5 bateaux et 2 plateaux, un pour placer les bateaux que l’autre ne voit pas et un autre pour tirer les missiles.

Une fois tous les bateaux placés, la partie peut commencer. Chaque joueur joue à tour de rôle et donne des coordonnées pour tirer sur les bateaux ennemis. Si un joueur touche un bateau, il peut de nouveau jouer.

La partie se termine quand un joueur fait couler toute la flotte ennemie.

Outils utilisé
  • Java
  • Github

Jeu de Dames

Année : 2023

Contexte

Projet réalisé sur une durée de 5 semaines. On devait réaliser un jeu de plateau en binôme sans le concept de programmation objet et avec comme seul interface le terminal. Utilisation de Github

A la fin de notre projet, nous devions rendre des tests JUnit ainsi qu’une petite documentation.

Lien du projet : Jeu de Dames

Description du jeu

2 joueurs s'affrontent sur un plateau de 10 cases en largeur et 10 cases en longuer (pion blanc VS pion noir).Chaque joueur possède 20 pions et peut les déplacer sur leur diagonale vers l'avant.

Le premier coup est toujours joué par les blancs. Les joueurs jouent à tour de rôle après chaque coup. Lorsqu’un pion effectue une prise, Il se rend alors sur la case suivante et enlève le pion adverse.

Lorsqu'il atteint la dernière rangée, le pion devient une dame et se déplace sur la diagonale jusqu'à qu'il tombe sur un obstacle. Une partie se finit quand un des 2 joueurs n'a plus de pion.

Outils utilisé
  • Java
  • Github

Site cactus Hostel

Année : 2023

Contexte

Ce projet en trio consistait à réaliser une page d'accueil en anglais pour un hôtel situé à Saint-Denis à partir d’une page minimaliste donnée.

Celle-ci visait à promouvoir des chambres d'hôtel dont une orientée sur un athlète populaire de notre choix en vue des Jeux Olympiques de Paris 2024.

Notre projet devait répondre à la contrainte d’éco-responsabilité en conséquence du réchauffement climatique.

Lien du projet : Cactus Hostel

Description du site

La page d'accueil est divisé en 4 sections :

  • La 1ère qui contient une barre de navigation ainsi que des aperçues sur l'hôtel.
  • La 2ème présente la chambre spécial athlète et ces caractéristique. Nous avons choisi de représenter la star montante de la NBA Victor Wembanyama
  • La 3ème nous montre où est situé le stade de Stade de France et comment s’y rendre par rapport à l'hôtel.
  • Enfin la dernière qui contient les contacts de l’hôtel ainsi que son adresse et de quelque lien externe

Outils utilisé
  • HTML
  • CSS
  • Javascript
  • Trello
